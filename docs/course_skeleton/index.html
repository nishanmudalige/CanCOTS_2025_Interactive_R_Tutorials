Note that revealjs presentations can be presented as HTML slides or can be printed to PDF for easier distribution. 🎉

Print/Save to PDF:

Reveal presentations can be exported to PDF via a special print stylesheet.

  1. Toggle into Print View using the E key (or using the Navigation Menu)
  2. Open the in-browser[^2] print dialog (CTRL/CMD+P).
  3. Change the Destination setting to Save as PDF.
  4. Change the Layout to Landscape.
  5. Change the Margins to None.
  6. Enable the Background graphics option.
  7. Click Save 🎉

Visit https://quarto.org/docs/presentations/revealjs/ for more details on create revealjs slides.
